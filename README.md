# 🏀投籃遊戲 Basketball Shooting Game
* 以第一人稱視角搭配腳步聲及投籃音效，提供玩家沉浸式體驗

## 實際介面
* 玩家可前後左右自由移動，以及向上跳躍
* 利用畫面中的準心，能夠更順利將球投入籃框
* 左上角的 `Point` 會在偵測到籃球進籃後更新計分
* 📢 注意：在球場外為無法投球的狀態！

## 遊玩操作
* `↑` `⭣` `←` `→` 分別表示往前、後、左、右方向移動
* 按下 `(space)` 可跳躍
* 點擊滑鼠左鍵將籃球投出
* 滑鼠的移動可改變鏡頭拍攝視角
